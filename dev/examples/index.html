Simulating Random Basis Functions

Load some packages that we will need and set a seed.

We start by simulating some random data from the model $Z = UDV' + \epsilon,$ where $Z$ is a n (space) by m (time) matrix, $U$ is a $n \times k$ matrix, $V$ is a $m \times k$ matrix, and $\epsilon \sim N(0, \sigma^2)$.

# set dimensions
n = 100
m = 50
k = 5

# domains
x = range(-5, 5, n)
t = range(0, 10, m)

# covariance matrix
ΣU = MaternKernel(x, ρ = 3, ν = 4, metric = Distances.Euclidean())
ΣV = MaternKernel(t, ρ = 3, ν = 4, metric = Distances.Euclidean())

# random n by k matrix with structure
Φ = PON(n, k, ΣU.K)
Ψ = PON(m, k, ΣV.K)
